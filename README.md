# 🗼 Ascend Tower


**Ascend Tower** es un plataformas de precisión 2D hecho en Godot Engine. Controlas a un protagonista que cae atrapado en el nivel más bajo de una torre legendaria y debe escalarla entera para escapar — piso a piso, trampa a trampa.

---

## 🎮 De qué va el juego

- Varios niveles de dificultad progresiva, cada uno representa un piso de la torre
- El objetivo de cada nivel es llegar a la moneda/cuerda que te sube al siguiente piso
- Tocar un pincho o una sierra significa **muerte instantánea** y vuelta al principio
- Al completar la campaña principal se desbloquea el **Modo Extremo**: versiones retocadas de cada nivel diseñadas para ser un dolor de cabeza (pero posibles de pasar)

## 🕹️ Controles y sensaciones

El movimiento está inspirado en **Super Meat Boy** — rápido, preciso y muy responsivo. El personaje tiene:

- Movimiento horizontal con aceleración
- Saltos de altura variable (pulsa poco para salto corto, mantén para salto largo)
- **Coyote time** (puedes saltar un instante después de salirte de una plataforma)
- **Deslizamiento por paredes y wall jump**

## 🎨 Estética

El juego tiene una **estética retro arcade** inspirada en clásicos como Castlevania y Donkey Kong. Un **shader VHS/CRT** personalizado cubre toda la pantalla añadiendo líneas de escaneo, estática, efecto ojo de pez y alto contraste. Los gráficos usan pixel art.

## 🔊 Banda sonora

Cada piso tiene su propia música generada con **Suno AI**, adaptada al ritmo y tensión de cada zona de la torre.
